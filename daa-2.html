Algorithm Description

Naive Algorithm

The Naive String Matching algorithm checks every possible position of the pattern inside the text by comparing characters one by one.

Rabin-Karp Algorithm

Rabin-Karp computes hash values of the pattern and text window. Only if both hashes match does it compare characters.

Knuth-Morris-Pratt (KMP)

KMP preprocesses the pattern by creating an LPS (Longest Prefix Suffix) array. It avoids unnecessary comparisons and provides linear-time searching.
